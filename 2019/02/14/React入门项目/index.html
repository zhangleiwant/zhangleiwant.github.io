#React.js项目
目前用的最多的前端框架。适用于复杂度较高的大型项目。vue.js 内部api比较多 实现功能简单，但是不便控制。
脚手架工具：
前端开发的辅助工具，会自动的帮我们构建前端开发流程和目录，它允许我们以一定的方式去实现js的交互引用。更方便的管理项目。 (GRUNT webpack gulp) 官方的脚手架：Create-react-app。

npm ：
node包管理器，先安装node.js ;node官网下载版本。
安装create-react-app 脚手架。
npm install -g create-react-app
创建react.js项目：
create-react-app my-app
cd my-app
运行服务，成功后可以再本地看到运行成功的页面。
npm start
